# 书籍数据源说明

## 已接入

### 1. Open Library（主数据源）
- **来源**：https://openlibrary.org
- **类型**：公开 API，无需 Key
- **用途**：搜索书籍、获取基本信息、封面
- **覆盖**：外文书籍丰富，中文书籍有限

### 2. 豆瓣读书（补充数据源）
- **来源**：https://api.douban.com/v2/book
- **类型**：公开 API，**无需 Key**
- **限制**：单 IP 约 150 次/小时
- **用途**：
  - 中文关键词搜索时合并结果，增加中文书籍
  - 有 ISBN 时补充评分、封面、简介
- **覆盖**：中文书籍丰富，评分、简介质量高

## 京东图书（占位）

- **状态**：未实现，需 API Key
- **说明**：京东开放平台需企业资质，配置 `JD_APP_KEY` 和 `JD_APP_SECRET` 后可扩展

## 数据流

```
init_books.py 执行时：
├── 中文关键词 → Open Library + 豆瓣 双源搜索 → 合并去重
├── 外文关键词 → 仅 Open Library
└── 保存前 → 有 ISBN 时用豆瓣补充评分/封面
```

## 注意事项

- 豆瓣 API 在国内可能需代理或延迟较高
- 若豆瓣请求失败，会静默降级，仅使用 Open Library 数据
- 运行 `python scripts/init_books.py` 时会自动使用豆瓣补充
